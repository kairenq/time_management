# Time Management — React приложение учета рабочего времени

Простое SPA-приложение на React для демонстрации требований из курсовых материалов по системе «Учёт рабочего времени»:

- авторизация и роли (администратор, руководитель, сотрудник);
- фиксация прихода/ухода и журнал явок;
- ведение графика смен/мероприятий;
- заявки на отпуск, больничный, отгул, командировку;
- табельный отчёт и базовая аналитика.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`.
